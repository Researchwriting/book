# Universal Content Generator

This is the central hub for the Universal Content Generator system.

## Project Structure

- **`books/`**: Contains the Textbook and Book Generation system.
- **`thesis/`**: (Coming Soon) Academic thesis and paper generation.
- **`main.py`**: The unified entry point to launch any tool.

## How to Run

Simply run the main script from this directory:

```bash
python3 main.py
```

Select **[1] Book Generator** to access the textbook generation tools.
